Descrição do Projeto

O Projeto Loja Ninja foi feito no intuito de praticar os conhecimentos adquiridos em HTML e CSS.
Nesse projeto, cada aluno poderia ficar à vontade para escolher sobre qual tema seria seu projeto-loja.
Eu fiz meu projeto baseado no anime Naruto, já que trabalho com coisas que nós conhecemos facilitam nossa produtividade.
Então, as ideias desenvolvidas aqui, são baseadas no universo do anime Naruto.
